# Functions
- Width changing
- Height changing
- Adaptive resize
- Resize retaining aspect ratio
- Quality compressing
- Convertion
  - WEBP
  - PNG
  - JPEG
  - Telegram sticker PNG format
- EXIF metadata editing
- Rotating
- Flipping
- Cropping
- Preset shrinking
